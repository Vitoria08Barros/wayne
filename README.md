# WayneIndustries
Meu projeto final de desenvolvimento full stack, intitulado “Wayne Industries”, um sistema web que simula um ambiente corporativo com controle de acesso e autenticação de usuários.

O sistema foi desenvolvido utilizando Python (Flask) no backend, MySQL como banco de dados e HTML, CSS e JavaScript no frontend. A aplicação conta com integração entre frontend e backend por meio de API REST, permitindo login e validação de usuários em tempo real.

O objetivo do projeto foi demonstrar a construção de uma aplicação completa, integrando autenticação de usuários, separação de níveis de acesso e comunicação entre camadas da aplicação.
